# Streamlit App Prompt for PHMbench

该文档整理了为 PHMbench (又称 vibench) 设计基于 Streamlit 的图形化应用时可提供给 AI 助手的提示词。本提示词旨在帮助生成一个直观、用户友好的界面，以便加载数据、选择并修改实验配置并执行完整的 Pipeline。

## 角色与目标
- **角色**：资深 AI 工程师，熟悉 Streamlit、PyTorch 与 MLOps。
- **目标**：将基于命令行和 YAML 配置的实验流程转化为图形界面，提高研究和调试效率。

## 核心用户体验
1. **选择数据模式**：提供传统文件上传与元数据两种方式。
2. **加载数据**：
   - 传统模式：用户上传训练、验证和测试信号文件。
   - 元数据模式：上传 `metadata.csv/xlsx`，指定 HDF5 数据目录，并通过 ID 划分数据集。
3. **加载基础配置**：从 `configs/` 目录选择 YAML 模板。
4. **交互式调参**：通过滑块、下拉框等组件修改关键参数（学习率、批大小、模型类型等）。
5. **执行与监控**：点击“开始实验”按钮后后台运行 vibench 流程，并实时展示进度。
6. **结果可视化**：展示指标、训练曲线与预测对比图等。

## 关键组件实现
- **侧边栏 (`st.sidebar`)**：用于配置数据加载和实验参数。
- **主页面**：用于启动实验并展示结果，可使用 `st.tabs` 划分指标、训练曲线等区域。
- **后端逻辑**：收集 UI 输入，整合为 Hydra 兼容配置对象，调用 vibench `Pipeline` 执行，并将结果回传前端。
